# 踢踢的水印小工具 🪄

> 写稿大王踢踢专用的文档守护神器

由 [twoheart](https://github.com/twoheartliu) ❤️ 为踢踢量身打造，让她的每一篇稿件都得到最好的保护。

完全在浏览器中运行，无需上传文件到服务器，文档隐私安全有保障。

## ✨ 功能

- **自定义水印文字**：下拉预设一键选择，也支持自由输入
- **水印样式定制**：
  - 自定义水印文字（预设：踢踢专用 / 写稿大王踢踢 / 严禁复制）
  - 颜色选择器支持任意颜色
  - 透明度、字体大小、旋转角度、水印密度灵活可调
- **中文字体支持**：内置黑体，完美呈现中文水印
- **本地处理**：全程在浏览器中完成，文件不会离开你的电脑
- **记忆偏好**：水印设置自动保存，下次打开无需重新配置
- **自适应布局**：桌面和移动端都能愉快使用

## 🛠️ 技术栈

- **Vue 3** + **Vite** — 响应式前端框架与极速构建
- **shadcn-vue** + **Tailwind CSS v4** — UI 组件与设计系统
- **Reka UI** — 无头可访问 UI 基元
- **Claude 设计风格** — 暖奶油色背景 + 珊瑚色主色调
- **pdf-lib** + **fontkit** — PDF 处理与中文字体嵌入
- **file-saver** — 浏览器端文件下载

## 📦 本地运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 🚀 使用

1. 上传 PDF 文件
2. 选择水印文字（预设或自由输入），调整颜色、透明度、大小等
3. 点击"✨ 添加水印"，自动下载带水印的 PDF

## 🔒 隐私

所有处理在浏览器本地完成，PDF 文件不会上传到任何服务器。
