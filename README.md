# 踢踢的水印小工具 🪄

> 给踢踢的每一份创作加上印记

由 [twoheart](https://github.com/twoheartliu) ❤️ 为踢踢量身打造，文档、图片都能保护。

## ✨ 功能

- **📄 文件水印**：支持 PDF 和 DOCX，统一输出带水印的 PDF
- **🖼️ 图片水印**：支持 JPG / PNG / WebP，保持原图格式，多图自动打包 ZIP
- **水印样式定制**：
  - 水印文字（预设：踢踢专用 / 写稿大王踢踢 / 严禁复制 / 心心写稿）
  - 颜色、透明度、字体大小、旋转角度、水印密度灵活可调
- **中文字体支持**：内置黑体，完美呈现中文水印
- **记忆偏好**：水印设置自动保存，下次打开无需重新配置
- **自适应布局**：桌面和移动端都能愉快使用

## 🛠️ 技术栈

**前端**
- **Vue 3** + **Vite** — 响应式前端框架与极速构建
- **shadcn-vue** + **Tailwind CSS v4** — UI 组件与设计系统
- **pdf-lib** + **fontkit** — PDF 水印处理与中文字体嵌入
- **Canvas API** — 图片水印渲染
- **JSZip** — 多图片 ZIP 打包

**后端**
- **Express** — HTTP 服务
- **LibreOffice** — DOCX → PDF 转换
- **PM2** — 进程保活

## 📦 本地运行

```bash
# 安装前端依赖
pnpm install

# 安装后端依赖
cd server && npm install && cd ..

# 一键启动前端 + 后端
pnpm dev:all
```

前端运行在 `http://localhost:5173`，后端运行在 `http://localhost:3001`。开发模式下 Vite 自动代理 `/api` 到后端。

## 🚀 使用

### 文件水印
1. 上传 PDF 或 DOCX 文件
2. 调整水印样式
3. 点击"✨ 一键添加水印"，自动下载带水印的 PDF

### 图片水印
1. 切换到 🖼️ 图片水印标签
2. 选择一张或多张图片
3. 调整水印样式，点击添加水印
4. 单张直接下载，多张打包为 ZIP

## 🔒 隐私

- **PDF / 图片水印**：全程在浏览器本地完成，文件不会离开你的电脑
- **DOCX 水印**：文件上传至后端服务器进行格式转换（LibreOffice），转换完成后即刻删除，不留存
